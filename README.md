
# R·CURVE — Professional Backtest Journal

A high-fidelity, visually striking frontend interface for a modern trading journal and backtesting dashboard. Built with a sleek neon-cyberpunk and glassmorphic aesthetic for Forex, Crypto, and Stock traders.

## ✨ Core Features

* **Cyberpunk Glassmorphism:** Deep space backgrounds with vibrant glowing accents and frosted-glass effects.
* **Dynamic Dashboard:** UI space mapped out for Win Rate, Net Profit, Profit Factor, and risk multiples (R).
* **Mobile Optimized:** Collapsible columns, an off-canvas side drawer, and a thumb-friendly bottom navigation bar.

## 🛠️ Tech Stack

* **HTML5:** Semantic structure.
* **CSS3:** Custom properties (variables), Grid/Flexbox, and complex keyframe animations.

## 🚀 How to Run

1.  Copy the code into a file named `index.html`.
2.  Double-click the file to open it in any modern web browser.

## 🚧 Status & Next Steps

This repository currently contains a **static visual layout only**. To make it a fully functional app, the following need to be built:

* **JavaScript Logic:** To handle login transitions, modal toggles, and form submissions.
* **Live Charts:** Integration of a library like `Chart.js` or `ApexCharts` to replace the central placeholder.
* **Data Persistence:** Connection to browser `localStorage` or a backend database so logged trades don't disappear on refresh.
